Sistema de alarme simples feito em Java, deve ser iniciado junto ao pc, seja manualmente por meio de um atalho ou automaticamente utilizando o agendador de tarefas do proprio computador.

O arquivo Alarmes utiliza o seguinte layout: NomedoAlarme,HorariodeInicio,HorariodeTermino. Não há um limite de alarmes, apenas certifique-se de seguir o padrão.
